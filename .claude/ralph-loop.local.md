# タスク指示書: タイムマネジメントWebアプリ「tikkle」初期構築

## 概要

タイムマネジメントWebアプリケーションを新規構築する。カウントダウン、経過時間表示、スタミナ回復追跡、定期増加リソース管理など、複数種別のタイマーを管理できるアプリケーション。

- ブランチ: 'master'（コミットなし、完全に空）
- プロジェクトファイル: 一切なし

## タイマー種別（5種）

1. **カウントダウン（countdown）** — 指定日時までの残り時間を表示（例: 賞味期限まであと何日）
2. **経過時間（elapsed）** — 指定時点からの経過時間を表示（例: 何かをしてからn時間n分経過）
3. **カウントダウン→経過切替（countdown-elapsed）** — 目標時刻まではカウントダウン、過ぎたら経過時間に自動切替
4. **スタミナ回復（stamina）** — 現在値/最大値 + 回復レート（例: n分ごとに1回復）→ 完全回復までのカウントダウン
5. **定期増加+上限（periodic-increment）** — FF14リーヴ券型: 上限値、増加量、増加スケジュール（例: 09:00/21:00に+3、上限100）→ 上限到達までのカウントダウン
